Hotkeys

 

This is a summary of all the hotkeys in this software:

F1 – Help.

CTRL + NUMBER – Focus onto a clip (Cycle Mode ON) or copy into/paste from the clip specified by the number (Cycle Mode OFF), where NUMBER can be any key from 1 to 5.

CTRL + INSERT – Toggle between COPY and PASTE mode.

INSERT – copy into/paste from the focused clip when Cycle Mode is ON.
