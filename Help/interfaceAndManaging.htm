Graphical Interface & Clipboard Management

 

The Graphical User Interface

There are 2 possible options for the graphical interface that you can choose from. The default one, called Clipboard Manager, shows all 5 clips, their contents, and all the modes. The other one, called QuickBoard, is more compact and can be seen as a minimal version of the main manager window.

- To go from Clipboard Manager to QuickBoard and vice versa, click on Options.

- To keep the window on top of the other windows, click on Stay on top, also in Options. This is recommended when you are working on one screen and want to keep track with the clipboard.

- Exit the software by either click on the X button at the top right corner of the window (either Clipboard Manager or QuickBoard) or choose Exit from the tray menu.

 

Clipboard Management

A Clip is one of the five available indexed text stored in the memory and really to be used any time you like.

- To make a Clip active, check on the box to the write of the Clip in the Clipboard Manager. To disable a clip, simply uncheck the box. When a clip is not active and Cycle Mode is on, the clip is skipped.

- To focus onto the clip in Cycle Mode (ON), simply press CTRL + NUMBER (where NUMBER can be any key from 1 to 5). Within the Clipboard Manager window, you can focus onto the clip by clicking the radio button on the left of the clip. The focused clip will have this button in blue. If using QuickBoard, the clip showed is always in focus (or last used when Cycle Mode is OFF); bring a clip into focus by clicking the button to the left of the clip, which will cycle through the active clips, incrementing by one at a time.

- In the Clipboard Manager, you can view the full text stored in the clip. To do so, click on expand. You can edit the text and save it from the popped up window.

- Click Clear All inside the Clipboard Manager to empty all the clips.
